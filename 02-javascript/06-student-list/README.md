# Student List

## Description

Stores student names inside an array and displays them dynamically.

## Features

- Add students
- Prevent empty names
- Prevent duplicate names
- Display total students
- Update student list automatically

## Concepts Used

- Arrays
- Loops
- DOM
- Event Listeners
- User Input

## Improvements

- Learned how to use for loops
- Learned duplicate checking
- Practiced updating the DOM

## Future Improvements
