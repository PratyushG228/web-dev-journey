# HTML & CSS

This folder contains my learning journey through the fundamentals of web development using **HTML5** and **CSS3**.

The goal of this phase was not to build perfect-looking websites, but to understand how webpages are structured, styled, and made responsive without relying on frameworks.

---

## 📚 Topics Covered

### HTML
- Basic document structure
- Semantic HTML
- Headings, paragraphs, links, images
- Lists
- Forms and inputs
- IDs, classes, and names
- Accessibility basics (labels, alt text)

### CSS
- Selectors
- Colors and typography
- Box Model
- Margin & Padding
- Borders and Border Radius
- Flexbox
- Responsive Design
- Media Queries
- Hover effects & transitions
- CSS organization and best practices

## 💡 Key Takeaways

- HTML provides the **structure**.
- CSS controls the **appearance**.
- Flexbox makes layouts much easier.
- Semantic HTML improves readability and accessibility.
- Responsive design is essential for modern websites.
- Clean, organized code is just as important as working code.
